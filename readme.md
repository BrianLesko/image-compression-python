
# K-means-clustering-colors-python
This code compresses an image to k colors using k means clusterng from the scikit-learn library. It isnt particularly fast. 

&nbsp;

<div align="center"><img src="docs/preview.gif" width="800"></div>

&nbsp;

## Dependencies

This code uses the following libraries:
- `streamlit`: for building the user interface.
- `scikit-learn`: for kmeans
- `numpy`: for arrays

&nbsp;

## Topics 
```
Python | streamlit | low code | UI | kmeans | scikit-learn | machine learning 
```
